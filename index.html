Contabilidad de valor

La contabilidad del valor es una forma de realizar un seguimiento de todas las contribuciones de valor a un proyecto.
Esto puede incluir todo tipo de valores que podamos definir como valiosos para nuestro proyecto: trabajo realizado, asesoramiento, recursos, dinero en efectivo...
Es útil visualizar cómo se debe hacer una división equitativa. También se puede usar para involucrar a las partes interesadas para seguir agregando valor.
Como contabilidad de valor, estamos hablando de acciones contables que no son acciones legales y que ayudamos a contabilizar y certificar usando tecnologías como los tokens fungibles y no fungibles para usos contables pero con validez interna dentro de una comunidad o equipo.
Timefounder ofrece la primera aplicación de contabilidad de valor que se integra con Trello.

La contabilidad de valor reduce el conflicto entre las partes de un proyecto evitando conflictos y ayudando a la distribución justa de las recompensas del mismo.
La app permite contabilizar las aportaciones que cualquier persona o entidad que aporte "valor" al proyecto.
